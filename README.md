## Overview

This project provides a hands-on introduction to the **7 Domains of Cybersecurity**, explaining the purpose of each domain and providing real templates, checklists, playbooks, and reference guides used in professional environments.

| Domain | Purpose |
|-------|---------|
| Governance | Direct security strategy, policies, accountability |
| Risk Assessment | Identify and mitigate vulnerabilities and threats |
| Physical Security | Protect buildings, equipment, and people |
| Security Operations | Detect and respond to cyber incidents |
| Security Architecture | Secure system and network design |
| Awareness & Training | Empower users to make secure decisions |
| Frameworks & Standards | Align with recognized security best practices |
